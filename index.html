Sobre Mim

Sou Sôlin Heusser, formando em Análise e Desenvolvimento de Sistemas. Durante a faculdade, busquemos entender não apenas o código, mas como um projeto funciona desde o planejamento inicial até o momento de colocar o sistema no ar.

No desenvolvimento prático, meu foco atual é o Backend com Python e FastAPI, utilizando o banco de dados SQLite. Para a parte de implantação de projetos, utilizo o GitHub Pages para publicar sites de forma automática e estou aprendendo a usar o Alembic para organizar as tabelas do banco.

Também tenho muito interesse nas áreas de Gestão de Projetos e Metodologias Ágeis, buscando entender como organizar tarefas e regras de negócio. Meu objetivo é criar soluções simples, organizadas e seguras que resolvam problemas reais do dia a dia.
